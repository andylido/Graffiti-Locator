Welcome to Graffiti Locator, this intuitive application allows users to search by ward and status of graffitis to help locate and track them down.

Instructions:
  • Input location information in the search tab.
  • The list tab will display: street address, service request number, graffati location, type of surface, date created, status and completion date (if applicable).
  • The map tab will utilize google maps api to locate all graffatis with marker pinpoint for further details.
